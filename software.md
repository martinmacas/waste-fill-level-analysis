Description:  
MATLAB-based software for analysis of fill-level signals from sensors deployed in public waste containers. The software enables detection of waste collection events using pattern classification and signal processing methods. It also identifies anomalies in the signal, such as blocked inlets, sudden drops, sensor faults, and missing or suspicious collections.  

The toolbox further supports inference of waste collection schedules, including periodic schedules (e.g., once every several weeks) and weekly schedules (e.g., specific weekdays). The methods are designed to work with both fast-filling waste streams (e.g., paper, plastic) and slow-filling waste streams (e.g., glass, metals).  

The software is intended for use in waste management optimization, smart city applications, and research scenarios, particularly in early-stage evaluation and test-before-invest analyses.
