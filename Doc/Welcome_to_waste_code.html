Dokumentace kódu pro EDIH Odpady

Tento dokument popisuje hlavní postup jakým použít MATLAB kód pro s časovými řadami zaplnění odpadových nádob.
Autor:* Martin Macaš Datum:* 2025-12-09 Verze: 1.0

Základní workflow:

Hlavní kroky využití kódu jsou:
  1. Načtení dat z API pomocí funkce api_read_measurements.
  2. Analýza signálu dle typu odpadu: analyse_signal pro papír, plast a rychle se plnící nádoby nebo analyse_signal_bsklo pro sklo a pomalu se plnící nádoby.
  3. Optimalizace svozů funkcemi pomocí optimize_collections_constraints pro rychle se plnící nádoby a optimize_collections_periodical pro pomalu se plnící nádoby.
