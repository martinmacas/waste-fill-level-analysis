api_read_measurements 

API_READ_MEASUREMENTS Načte měření zaplnění kontejneru z Golemio API.

   [Time, p, measurements] = API_READ_MEASUREMENTS(ksnko_id, apiKey)
   načte data o zaplnění kontejneru z Golemio Smart City API.
   Funkce dotazuje endpoint:

       https://api.golemio.cz/v2/sortedwastestations/measurements

   VSTUPY:
       ksnko_id : Číselný identifikátor kontejneru v systému Golemio.
       apiKey   : API klíč (token), který musí být předán v HTTP hlavičce.

   VÝSTUPY:
       Time         : Pole typu datetime (UTC). Obsahuje časová razítka
                      jednotlivých měření (pole 'measured_at_utc').

       p            : Číselné pole hodnot zaplněnosti (percent_calculated) pro každé měření.

       measurements : Struktura (nebo pole struktur) obsahující kompletní
                      nezaformátovanou odpověď API vrácenou funkcí WEBREAD.

   POZNÁMKY:
       • V případě chyby připojení, timeoutu nebo chyby API funkce vrací
         prázdné výstupy: Time = [], p = [], measurements = [].
       • Funkce používá 30sekundový timeout a předává API klíč v hlavičce
         'X-Access-Token'.

   PŘÍKLAD:
       ksnko_id = 12345;
       apiKey   = 'VAŠE_GOLEMIO_API_KEY';
       [T, P, data] = api_read_measurements(ksnko_id, apiKey);
